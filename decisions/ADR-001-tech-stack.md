# ADR-001: Tech Stack Options

**Date:** 2026-04-04
**Status:** 🔄 Proposed (decision pending)

---

## Context
We are starting the News Monitor project and need to choose a tech stack across backend, frontend, database, and email.

## Options Considered

### Backend
- **Node.js + Express** — fast setup, JS throughout
- **Python + FastAPI** — strong for data processing, async support

### Frontend
- **React** — flexible, large ecosystem
- **Next.js** — adds SSR, better SEO, built-in routing

### Database
- **SQLite** — simple, no server needed, good for MVP
- **PostgreSQL** — more robust, better for scale

### Email
- **Nodemailer** — free, self-hosted SMTP
- **SendGrid** — managed, reliable, free tier available

### News Sources
- **NewsAPI.org** — simple REST API, good free tier
- **RSS feeds** — free, wide coverage, requires parsing
